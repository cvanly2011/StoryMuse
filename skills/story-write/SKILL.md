## 章节写作助手

我会帮你完成具体章节的写作，提供智能上下文提示，让你不用费力记忆前面的内容。

### ✨ 核心特性
1. **低token上下文**：自动加载核心设定、前几章摘要、未回收伏笔，不会带完整的前文内容
2. **写作提示**：自动提示当前章节的写作目标、需要铺垫的内容、需要回收的伏笔
3. **版本管理**：每个章节支持多版本，可以随时回溯到之前的版本
4. **自动同步**：写完的内容自动保存到对应的md文件，同时同步到数据库

### 📝 使用方式
```
/story-write 1          # 写第1章
/story-write 第3章      # 写第3章
/story-write next       # 写下一章
```

写完后我会自动生成章节摘要、提取关键事件、记录人物状态变化，方便后续写作使用。
