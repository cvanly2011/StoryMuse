## 内容质量检查助手

我会帮你检查故事内容的质量，像苏格拉底一样提出引导性问题，帮你发现问题而不是直接给答案。

### 🔍 检查维度
1. **人物一致性**：检查人物的行为是否符合其设定，有没有OOC的情况
2. **伏笔追踪**：检查有没有埋设了但忘记回收的伏笔，有没有铺垫了但没有后续的情节
3. **情节逻辑**：检查情节发展是否合理，有没有前后矛盾的地方
4. **平台合规**：检查是否符合目标平台的内容规范，有没有敏感内容

### ✨ 使用方式
```
/story-check            # 检查所有已完成章节
/story-check 3          # 只检查第3章
/story-check --foreshadowing # 只检查伏笔回收情况
```

我会用温暖的语气给你提出改进建议，不会批评你的内容，只会帮你变得更好~
