## 安装说明
### 方式一：Claude Code插件市场安装（推荐）
1. 打开Claude Code → 设置 → 插件
2. 搜索「StoryMuse」
3. 点击安装即可自动完成所有配置

### 方式二：手动安装
1. 下载最新版本的插件包
2. 打开Claude Code → 设置 → 插件 → 「从本地安装」
3. 选择下载的zip包即可

## 快速开始
### 30秒创建你的第一部小说
1. 新建一个空文件夹，用Claude Code打开
2. 输入`/create-novel "我的第一部小说" "我的故事核心创意"`
3. 选择你的目标发布平台
4. 等待几秒钟，你的写作项目就创建完成了！

创建完成后，工作区会自动生成以下文件：
```
你的项目/
├── .claude/                  # 插件数据目录（自动管理，不用手动修改）
├── .story-muse.config.json   # 项目绑定配置（自动生成）
├── story-seed.md             # 故事核心设定
├── outline.md                # 故事大纲
├── characters.md             # 人物设定
└── chapters/                 # 章节内容目录
    └── 第1章 章节标题.md
```

## 核心功能
### 🎯 7个核心命令
1. **/story-seed**：梳理故事核心设定
2. **/story-skeleton**：生成三幕式大纲结构
3. **/story-characters**：完善人物设定和关系网络
4. **/story-write <章节号>**：开始写具体章节
5. **/story-check [章节号]**：检查内容质量和合规性
6. **/story-map**：查看全局创作进度
7. **/load <参数>**：同步手动修改的文件到数据库

### ✨ 核心特性
1. **完全本地存储**：所有内容都存在你的电脑上，不上云，绝对安全
2. **低token优化**：写作时只加载核心上下文，不传递完整前文，节省token
3. **多平台适配**：自动适配起点、晋江、番茄等主流网文平台的创作规则
4. **智能检查**：自动检查人物一致性、伏笔回收、情节逻辑问题
5. **Git友好**：支持每个分支对应一个故事版本，方便多结局尝试
6. **直接发布**：章节内容都是标准Markdown文件，直接复制就能发布到平台

## 工作流指南
### 标准创作流程
1. **创意阶段**：使用`/story-seed`梳理你的核心创意
2. **大纲阶段**：使用`/story-skeleton`生成完整的三幕式大纲
3. **人物阶段**：使用`/story-characters`完善人物设定和关系网络
4. **写作阶段**：使用`/story-write`逐章创作，写完自动同步到文件
5. **检查阶段**：使用`/story-check`检查内容质量，优化问题
6. **全局查看**：随时使用`/story-map`查看创作进度

### 手动修改文件流程
你可以随时直接编辑工作区的任意md文件，修改完成后：
```
/load --modified  # 自动同步所有修改过的文件
```
系统会自动更新所有相关数据，不需要其他操作。

## Git分支使用指南
StoryMuse完美适配Git工作流，你可以用不同分支管理不同的故事版本：
1. **尝试不同大纲**：新建分支修改大纲，不满意直接删除分支
2. **多结局创作**：不同分支写不同结局，自由切换
3. **多平台适配**：不同分支适配不同平台的规则
4. **协同创作**：多人协作时，每个人在自己的分支创作，合并后使用`/load --all`同步

### 使用方法
1. 正常使用Git切换分支
2. 切换分支后系统会自动检测，提示你是否同步最新内容
3. 执行`/load --all`即可重建当前分支的数据库，完全恢复到该分支的内容状态

注意：不需要把`.claude`目录提交到Git，只需要提交工作区的md文件即可。

## 常见问题
### Q：重装插件会丢失我的创作内容吗？
A：不会，所有创作内容都存在你的工作区的md文件里，重装插件后执行`/load --all`即可完全恢复。

### Q：我可以在其他编辑器里修改内容吗？
A：完全可以，修改完成后回到Claude Code执行`/load --modified`即可同步。

### Q：支持百万字级别的长篇创作吗？
A：完全支持，我们做了很多性能优化，百万字级别也能流畅运行。

### Q：会上传我的创作内容吗？
A：绝对不会，所有内容100%存储在你的本地电脑，我们不会上传任何内容到服务器。

### Q：支持哪些平台的规则适配？
A：目前支持起点、晋江、番茄、七猫、纵横、17K、豆瓣阅读等主流网文平台。
